# Pass Cloudflare Check on RubyOnRemote

Проект на Ruby для автоматизованого обходу захисту Cloudflare на сайті `rubyonremote.com` за допомогою Playwright.

## 🚀 Основні можливості
- **Chrome Channel**: Використання реального браузера Google Chrome замість Chromium.
- **Persistent Context**: Збереження профілю браузера (Cookies, Cache) для імітації реального користувача.
- **Advanced Stealth**: Маскування ознак автоматизації (`navigator.webdriver`, WebGL, Plugins).
- **Human Activity**: Емуляція рухів миші, скролінгу та затримок.

## 📁 Структура проекту
- `test.rb` — Основний робочий скрипт для входу на сайт.
- `check_bot.rb` — Діагностичний скрипт для перевірки рівня анонімності на `bot.sannysoft.com`.
- `cloudflare_bypass_guide.md` — Детальна документація про те, як працює обхід Cloudflare.
- `browser_profile/` — Папка з даними вашого браузерного профілю (кеш, куки).
- `success.png` — Скріншот результату успішного входу.

## 🛠 Налаштування та запуск

### 1. Встановлення залежностей
Переконайтеся, що у вас встановлено Ruby та Google Chrome, потім виконайте:
```bash
bundle install
npx playwright install chromium
```

### 2. Запуск основного скрипта
```bash
ruby test.rb
```
Скрипт спробує зайти на сайт, дочекається проходження перевірки Cloudflare та збереже скріншот.

### 3. Діагностика (якщо виникають проблеми)
Якщо Cloudflare почав вас блокувати, запустіть перевірку анонімності:
```bash
ruby check_bot.rb
```
Після завершення перевірте файл `bot_check.png`, щоб побачити, які саме параметри детектуються як бот.

## ⚠️ Важливо
- Папка `browser_profile/` додана в `.gitignore`, щоб не завантажувати гігабайти кешу в репозиторій.
- Для локальної розробки в `test.rb` можна змінити `headless: true` на `false`, щоб бачити процес у вікні.
